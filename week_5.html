slicing in numpy

In NumPy we can access the portion of an arrav using the slice operator

1-D Array Slicing

Syntax: array[start: stop:step]

start : index of the first element to be included in slice

stop : index of the last element (exclusive)

step: step size between each element in the slice

TF (t)= total number of occurrence of term t in sentence (doc)/ total length of words in sentence (doc)

(doc)/ total length of words in sentence (doc)

IDF(t) = log ( total sentence (doc) length) / no of documents contains term t

tf-idf score = tf * idf

import numpy as np

# Given data
prices = [10, 12, 13, 12, 16, 15]
sales = [40, 38, 43, 45, 37, 43]

# Calculate sums
n = len(prices)
sum_x = sum(prices)
sum_y = sum(sales)
sum_x_squared = sum(x**2 for x in prices)
sum_xy = sum(x * y for x, y in zip(prices, sales))

# Calculate slope (m)
m = (n * sum_xy - sum_x * sum_y) / (n * sum_x_squared - sum_x**2)

# Calculate intercept (b)
c = (sum_y - m * sum_x) / n

# Predict sales when price is 20
price_to_predict = 20
predicted_sales = m * price_to_predict + c

print(f"Predicted sales when price is {price_to_predict}: {predicted_sales}")
